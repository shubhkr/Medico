# medico

A [SailsJS](http://sailsjs.org) application.

* NodeJS
* SailsJS
* ReactJS
* React Hot Loader
* React-Router
* Redux *
* Material UI
* Webpack
* Mocha
* Ghoulies


## Installation
---

Install <a href="https://github.com/ryuone/nenv">nenv</a> and build `nodejs`:

```
nenv install 6.9.1
nenv local v6.9.1
```

Install prerequisites:

```
npm install
```

Build and run development server:

```
npm run dev
```

Build and run production server:

```
npm run prod
```
